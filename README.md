# Ramendra Bharadwaj — Professional Portfolio

A premium, interactive, and responsive developer portfolio showcasing work experience, education, projects, publications, patents, and certifications.

## 🔗 Live Site
View the live portfolio here: **[https://bh-ramendra.github.io/Ramendra_Portfolio/](https://bh-ramendra.github.io/Ramendra_Portfolio/)**

---

## ✨ Key Features & UX Polish

*   **Premium Glassmorphism Design**: Standardized on a dark/light mode harmonized color palette, complete with subtle glowing background graphics and layout outlines.
*   **Dynamic Hero Stats Cards**: Features counters representing years of experience, corporate awards, and total verified certifications.
*   **Technical Expertise Marquee**: An infinite horizontal scrolling marquee of languages, frameworks, databases, and AI tools, complete with pause-on-hover actions.
*   **Visual TIMELINE Layouts**: Work history and education sections structured as visual timelines, supporting document attachments and image proofs.
*   **3D Perspective Projects**: Unified project showcase using custom 3D perspective boxes that tilt and lift on mouse hover.
*   **Professional Certifications Grid**: Direct link mappings for high-visibility credentials linking to verification details.
*   **Merged Recognition Carousel**: Auto-rotating recommendations and corporate awards carousel supporting manual controls, indicators, and mobile **touchswipe gestures**.
*   **3D Publication & Patent Covers**: Dedicated section for publications and patents featuring physical CSS-based 3D book covers, blueprints, and legal seals.
*   **Forms Intercept & Formspree AJAX**: Seamless integration with Formspree for the Contact form, Meeting Scheduler, and Newsletter sub-form, returning interactive glassmorphic inline success cards.
*   **Smart Scroll Utilities**:
    *   **Scrollspy**: Navigation items in the header highlight dynamically based on the active viewport section.
    *   **Auto-Hiding Search**: The header search bar collapses on scroll (shrinking from a two-row mobile header to a single clean line).
    *   **Scroll-to-Top**: A floating back-to-top button fades in smoothly below 400px scroll depth.
*   **Performance Optimized**: Image lazy loading (`loading="lazy"`) enabled on all offscreen assets below the fold.

## 🛠️ Technology Stack
*   **Structure**: Semantic HTML5 markup
*   **Styling**: Pure CSS3 variables & animations (Vanilla CSS)
*   **Logic**: Pure Vanilla JavaScript (ES6+)
*   **Icons**: Devicons & custom inline SVGs
*   **Hosting**: GitHub Pages

## 📂 Project Structure
*   `index.html` — Site layout and structural wrappers
*   `style.css` — Core design tokens, light/dark theme properties, responsive media rules, and custom 3D animations
*   `script.js` — Configuration arrays, dynamic render loops, carousel timers, touch events, and search index algorithms
*   `assets/` — Local image assets, profile photos, and badges
